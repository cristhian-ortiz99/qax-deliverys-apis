# Proyecto de Automatización de APIs con Playwright

## Descripción
Este proyecto contiene pruebas automatizadas de APIs utilizando Playwright con JavaScript.  
El objetivo es validar distintos endpoints mediante pruebas funcionales, asegurando el correcto comportamiento del sistema.

---

## Alcance de pruebas

Se automatizaron los siguientes flujos:

-  Login de usuario
-  Consulta de perfil de usuario
-  Recuperación de contraseña (Forgot Password)
-  Envío de token de recuperación por correo
-  Validación del token de recuperación
-  Cambio de contraseña (usuario autenticado)

---

## Tecnologías utilizadas

- Node.js
- Playwright
- JavaScript
- Postman (para pruebas manuales)
