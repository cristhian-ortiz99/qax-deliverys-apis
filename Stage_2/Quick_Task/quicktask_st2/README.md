# Playwright API Tests

Quicktask del Stage 2 - pruebas automatizadas de API utilizando Playwright con JavaScript.

---

## Descripción

Este proyecto contiene pruebas automatizadas para validar un endpoint de una API, incluyendo:

- Login de usuario
- Validaciones de respuestas (status, mensajes)

La validación se realiza y se muestra el ID y correo generado que se obtiene de la respuesta de la Api.
---

## Tecnologías utilizadas

- Node.js
- Playwright
- JavaScript
