# API Automation - Challengue_2

## Descripción

Flujo completo de ciclo de vida de un producto, utilizando las peticiones GET, POST, PUT y PATCH. En la implementación de este ejercicio se utilizaron tanto models como services para el mantenimiento correcto de nuestro código.

---

# Tecnologías utilizadas

- Node.js
- Playwright
- JavaScript
- Dotenv

---

# Instalación

## 1. Clonar repositorio

```bash
git clone <url-del-repo>
```

---

## 2. Instalar dependencias

```bash
npm install
```

---

## 3. Instalar navegadores de Playwright

```bash
npx playwright install
```

---

# Ejecución de pruebas

## Ejecutar todos los tests

```bash
npx playwright test
```

---

## Ejecutar un archivo específico

```bash
npx playwright test tests/product_lifecycle.spec.js
```
