# API Automation - Challengue_1

## Descripción

Flujo completo de actualización con PUT y PATCH, la implementación de esté fragmento de pruebas se encuentra debajo del comentario Challengue 1

---

# Tecnologías utilizadas

- Node.js
- Playwright
- JavaScript
- Dotenv

---

# Instalación

## 1. Clonar repositorio

```bash
git clone <url-del-repo>
```

---

## 2. Instalar dependencias

```bash
npm install
```

---

## 3. Instalar navegadores de Playwright

```bash
npx playwright install
```

---

# Ejecución de pruebas

## Ejecutar todos los tests

```bash
npx playwright test
```

---

## Ejecutar un archivo específico

```bash
npx playwright test tests/products.spec.js
```
